# API - Carga Pesada
API RESTful de notas para caminhões. É um serviço que permite o cadastro e gerenciamento de notas fiscais para cada parada que um caminhão faz durante uma viagem, especialmente quando realiza compras em diferentes cidades. Essa API é fundamental para manter um registro organizado e seguro das transações realizadas ao longo do percurso.

Desenvolvido com as seguintes tecnologias:
- Conceitos de boas práticas e qualidade no código, usando Design Patterns, Clean Architecture, Domain Driven Design (DDD) e Princípios SOLID
- PHP
- Laravel
- Swagger
- Banco de dados relacional **MySQL**
- **Docker**
